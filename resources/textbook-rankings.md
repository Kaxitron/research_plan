# Linear Algebra Textbook Rankings

| Resource | Strength | Best For | Free? |
|----------|----------|----------|-------|
| **3Blue1Brown (Essence of LA)** | Geometric intuition, animations | First-pass understanding | ✅ YouTube |
| **MIT OCW — Gilbert Strang** | Depth, four subspaces framework | Deep understanding, proofs | ✅ YouTube + MIT |
| **MML Book (Deisenroth)** | ML-focused notation & examples | Bridging to ML papers | ✅ Online |
| **Interactive Linear Algebra (GT)** | Embedded interactive demos | Playing with concepts | ✅ Online |
| **Immersive Linear Algebra** | 3D interactive visualizations | Spatial intuition | ✅ Online |
| **Steve Brunton's channel** | Applications-heavy (data science) | SVD/PCA/applied intuition | ✅ YouTube |
| **Strang's textbook** | Definitive reference | Going deep on any topic | ❌ Purchase |
| **BetterExplained** | Intuition-first explanations | Quick "aha" moments | ✅ Online |
| **Khan Academy** | Worked examples, practice | Mechanical fluency | ✅ Online |
| **Professor Leonard** | Thorough step-by-step | Building computation skills | ✅ YouTube |
